 <h1>프로젝트 Intranet</h1>
 <br>
 <h2>1. 프로젝트 명</h2>
 <h3>  회사 간단 그룹웨어 Intranet</h3>
 <br>
 <h2>2. 개발 목적 및 프로젝트 개요</h2>
 <h3>1) 개발 목적</h3>
 <h4>자바 프로그램밍 언어와 오라클 DB를 이용한 간단한 소프트웨어 개발 연습</h4>
 <h4>팀의 공동개발 방법 습득을 위해 업무 분장 및 개발 진행방법 습득</h4>
 <h4>Github 사용법 습득 및 활용</h4>
 <h3>2) 개발 인원 : 4명</h3>
 <h3>3) 개발 기간 : 2019.11.28 - 2019.12.06(10일)</h3><br>
 <h2>3. 개발 환경</h2>
 <h3>1) 사용 언어 : JAVA</h3>
 <h3>2) 사용 툴 : Eclipse, SQLDevelopter, MySQL Workbench, Netbeans</h3>
 <h2>3. 구현 기능 및 핵심 기술</h2>
 <h3>1) 주요 구현 기능</h3>
 <h4>로그인<br>
     사원 등록<br>
     사원 정보 수정<br>
     공지게시판<br>
       글 등록 및 삭제 (CRUD)<br>
	   한줄게시판<br>
	    - 글 등록 및 삭제 (CRUD)<br>
	    - 댓글 등록 및 삭제 (CRUD)<br>
	   메시지기능<br>
	 	  - 보낸/받은 쪽지List<br>
	 			-- 검색기능(사원번호, 사원이름, 내용)<br>
	    - 메시지 쓰기<br>
	 		 -- 사원 검색 기능
	    - 메시지 읽기<br>
	    - 메시지 삭제<br></h4>
  <h3>2) 핵심 기술</h3>
  <h4>JFrame UI 구현<br>
     Database 설계<br>
     JDBC를 이용한 Database 연동<br></h4><br>
